# **野村工務店 在庫管理システム (Nomura Tree Inventory Management)**

このリポジトリは、Google Apps Script (GAS) を利用して構築された、Webアプリケーション形式の在庫管理システムです。

Google スプレッドシートをデータベースとして使用し、ブラウザから直感的に在庫の登録・確認・売上管理を行うことができます。

## **🌲 システムの概要**

植木や関連商品の在庫状況を可視化し、日々の業務効率を向上させることを目的としています。専用のアプリをインストールする必要はなく、ブラウザ（Chromeなど）があればどこからでもアクセス可能です。

### **主な機能**

* **ダッシュボード (template-dashboard)**: 全体の状況を一目で確認できます。  
* **商品管理**:  
  * 商品一覧 (template-product-list)  
  * 商品詳細 (template-product-detail)  
  * 新規登録・編集 (template-product-form)  
* **在庫管理 (template-inventory)**: 現在の在庫数の確認や調整を行います。  
* **売上管理 (template-sales-list)**: 売上履歴の確認を行います。  
* **レポート (template-reports)**: データの集計や分析結果を表示します。  
* **マスタ管理 (template-master)**: カテゴリや設定値などの基本データを管理します。

## **🛠 技術スタック**

このシステムは以下の技術で構築されています。

* **バックエンド**: Google Apps Script (GAS)  
* **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla JS)  
* **データベース**: Google スプレッドシート  
* **通信**: google.script.run (非同期通信)

## **📂 ファイル構成**

プロジェクト内の主要なファイルの役割は以下の通りです。

| ファイル名 | 分類 | 説明 |
| :---- | :---- | :---- |
| Code.gs | Server | サーバーサイドのロジック。スプレッドシートとの通信やHTMLの配信を担当。 |
| index.html | Client | アプリケーションの全体枠（シェル）。ここに各画面が読み込まれます。 |
| styles-\*.html | CSS | デザイン定義。layout（配置）、components（部品）、common（共通）に分割されています。 |
| scripts-\*.html | JS | フロントエンドのロジック。api（通信）、app（アプリ動作）、common（共通関数）。 |
| template-\*.html | View | 各画面のHTMLテンプレート（ダッシュボード、在庫画面など）。 |
| appsscript.json | Config | GASプロジェクトのマニフェストファイル（設定ファイル）。 |

## **🚀 セットアップとデプロイ手順**

このシステムをご自身のGoogle環境で稼働させるための手順です。

### **1\. 準備**

1. データを保存するための新規 **Google スプレッドシート** を作成してください。  
2. スプレッドシートのメニューから 拡張機能 \> Apps Script を開き、エディタを起動します。

### **2\. コードの反映**

このリポジトリにあるファイルを、Apps Scriptエディタ内の同名ファイルにコピー＆ペースト（または clasp ツールを使ってプッシュ）してください。

**注意**: ファイルを作成する際、拡張子に注意してください。

* .gs は「スクリプト」  
* .html は「HTML」  
  として作成します。

### **3\. デプロイ**

1. Apps Scriptエディタ右上の「デプロイ」ボタンをクリックします。  
2. 「新しいデプロイ」を選択します。  
3. 種類の選択で「ウェブアプリ」を選びます。  
4. 以下の設定を行います：  
   * **説明**: （例：初回リリース）  
   * **次のユーザーとして実行**: 自分  
   * **アクセスできるユーザー**: 全員（または組織内のユーザー）  
5. 「デプロイ」をクリックし、発行された **ウェブアプリのURL** にアクセスするとシステムが利用できます。

## **⚠️ 注意事項**

* **権限について**: 初回アクセス時、Googleアカウントへのアクセス権限（承認）を求められる場合があります。「詳細」→「安全ではないページに移動」から承認を行ってください。  
* **スプレッドシートID**: Code.gs 内でスプレッドシートのIDを指定する箇所がある場合は、手順1で作成したシートのIDに書き換えてください。

## **🤝 開発者向け情報**

ローカル環境で開発を行い、clasp を使用してGASへアップロードすることを推奨します。

\# クローン  
git clone \<repository-url\>

\# 依存関係のインストール (必要な場合)  
npm install

\# GASへのプッシュ  
clasp push
